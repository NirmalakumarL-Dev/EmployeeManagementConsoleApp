# Employee Management Console App

## Description
A C# console application demonstrating:
- OOP principles (Encapsulation, Inheritance, Polymorphism)
- Interfaces & Abstraction
- Collections & LINQ
- Exception handling
- File-based persistence

## Features
- Add employee
- View employees
- Data saved to file

## Tech Stack
- C#
- .NET
- Console Application

## Purpose
Built as part of a structured learning path to transition into .NET backend development.
